Proiectul este realizat in CodeBlocks.
Pentru a lucra cu SDL2,am descarcat unele fisiere ale librariei grafice,de pe website-ul https://www.libsdl.org/download-2.0.php  
In repository,folderul SDL contine acele fisiere.
Am configurat CodeBlocks astfel incat sa pot folosi elementele librariei grafice.
Pentru a realiza acest lucru,am folosit instructiunile de pe urmatoarea pagina web: http://www.grhmedia.com/SDL001.html
La aceasta adresa pot fi gasite informatii privitoare la configurarea SDL2,inclusiv a unor librarii pe care nu le-am utilizat (SDL2_mixer,SDL2_image,SDL2_ttf).
Pentru program este necesar si un fisier .dll,pe care l-am inclus in proiect si care poate fi descarcat de pe https://www.libsdl.org/download-2.0.php
